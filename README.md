# 오픈소스 소프트웨어(8조)
주제: 퀴즈 풀이 프로그램

#기능 소개


    1) 게임 플레이어 모드
        - 문제 카테고리 선택
        - 문제 풀이
        - 문제 추가
        - 점수 및 랭킹 확인
    
    2) 관리자 모드
        - 플레이어가 추가할려고 하는 문제 확인
        - 플레이어의 문제 승인/거절
        - 기존의 문제 수정 및 삭제
        - 플레이어 유저 관리

    3) 기본적인 게임 기능
        - 타이머(제한시간) 기능
        - 플레이어가 요청하면 힌트 제공
        - 카테고리마다 등수 추가
